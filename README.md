# SmartStock_inventory_optimization_dashboard
Optimizing Inventory for a Small Retail Store
Project statement:
small retail stores often struggle with managing stock effectively due to limited forecasting capabilities and lack of automation. Overstocking leads to wastage or dead stock, while understocking causes lost sales and dissatified customers. This project uses machine learning and time series forecasting to predict product demand,enabling optimal restocking decisions. A visual dashboard helps the store owner track inventory health, suggested reorder levels, and stock anomalies.
